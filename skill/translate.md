# テキストの翻訳を行う

## スキルの手順

1. ユーザーが翻訳を求めたら「翻訳モード」に入る
    - 「翻訳を開始して」「今から言うことを翻訳して」などの発言で翻訳モードに入る
2. 翻訳モードの間、ユーザーの発言をただ翻訳して返す
    - この間、他のスキルは発動しない
    - SLEEP にも入らない
3. ユーザーが翻訳の停止を求めたら、翻訳モードを終了する
    - 「翻訳を停止して」「翻訳をやめて」「翻訳を終わりにして」など
2. 明示的に翻訳の停止を求められるまで、このスキルは発動を続ける

## 翻訳フォーマット

```
${JP_TEXT (日本語)}
${ZH_TEXT (中国語)}
${EN_TEXT (英語)}
[TRANSLATE]
```

### EXAMPLE

User: 「こんにちは、世界！」

```
こんにちは、世界！
你好，世界！
Hello, World!
[TRANSLATE]
```

User: 「次を翻訳して『怎么了？』」

```
いかがですか？
怎么了？
What's wrong?
[TRANSLATE]
```

User: 「Good night!」

```
おやすみなさい！
晚安！
Good night!
[TRANSLATE]
```

User: 「翻訳を停止して」

スキルを停止する
